Free text field

Unlike most Q-Action text, which is formatted for you, this field's text you are free to format yourself, using web formatting commands. They look scary but are quite easy: just take a deep breath and follow the guidelines below.

Each formatting command consists of some characters enclosed in angled brackets. You'll find these angled brackets hiding in your PC keyboard - probably bottom right, but phones don't usually have them at all so you may have to input them through a menu as special characters, or copy and paste them from this page.

A formatting command looks like this <h1>. These are known as tags. Each format command has two tags, one to show where it starts, and one with a slash added to show where it stops. For example, to display a price in bold you might put <strong> £9.99 </strong> .

The line break tag <br /> is a singleton as it is not really an on or off switch - it just starts a new line. You need to use this tag instead of pressing enter in the text editor. Although the enter key may appear to put your edited text on a new line, this won't be recognised by the user's browser.

The commands you can use are:

<h1> = start heading style 1

</h1> = end heading style 1

<h2> = start heading style 2

</h2> = end heading style 2

<strong> = switch on bold type

</strong> = switch off bold type

<br /> = Line break (carriage return)

If you are feeling adventurous you can create a bullet list. You have to start the list with the <ul> tag then start and end each line with the <li> and </li> . Finally you end the list with </ul>.

Here is an example:

Code

<h1>Corner Cafe</h1> <br />

<h2>Offer of the week</h2> <br />

Buy one get <strong>two</strong> free on the following: <br />

<ul>

<li>Capuccino Grande</li>

<li>Skinny Latte</li>

<li>Americano</li>

</ul> <br />

Offer ends Friday

Result

Corner Cafe


Offer of the week


Buy one get two free on the following:
  • Capuccino Grande
  • Skinny Latte
  • Americano

Offer ends Friday
