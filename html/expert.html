What are expert features?

Q-Action has been designed to work on almost any mobile device using a simple interface. These pages provide a range of more advanced features that some users may find useful. However, not all of these features will work on all devices. For example, file upload and download may not work on your mobile device, but it should work on almost any laptop or desktop computer with almost any web browser.

Download csv file

You may download a CSV version of your Q-Action pages that may be edited by almost any spreadsheet software. Supported software includes: MicrosoftOffice, LibreOffice, OpenOffice, NeoOffice and many other text editors such as Gedit. With some constraints, you may edit the file and upload all your changes as a batch.

Hints

If editing with a spreadsheet and your software does not automatically detect the required options try using the settings illustrated in the screenshot:

  • Unicode (UTF-7)
  • Separated by a comma ,
  • Text delimiter single quote '
  • If your import facility gives you a preview it should be readable. Note the single quotes round each entry and the underscores in the field names 'key_string' and 'auto_forward'. If either are missing then your settings are probably incorrect.

Upload csv file

You may also upload an edited version of the csv file. The following guidelines apply:

  • You may not edit the contents of the key_string column.
  • Any entry that has a comma must be enclosed in single quotes.
  • Many uploading errors will be cured by enclosing the entry in single quotes.
  • You may delete any row except the first row which must contain the field names.
  • You must leave at least one row of data.
  • All other columns are optional and may be deleted or re-ordered.
  • When uploading an edited file you must be signed in. Your file is unique to you and will not work on any other account.
  • Q-Action will confirm the success (or failure) of each uploaded row of data.
  • This useful facility is labelled as "for expert use only" for a reason. It really is a good idea to try uploading a few simple changes to spare cards to check your basic settings before uploading large numbers of changes.
