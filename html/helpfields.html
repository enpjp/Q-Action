Select Card Type

This pre-formats the card for a specific purpose. You can change type at any time. The Blank card gives you an unstructured page to lay out yourself.

Top

Card Title

A unique name you want to give this card for your own purposes. It will not appear to end users, but it will be used in the ‘My Cards’ list which will be sorted alphabetically by this name.

Top

Label for Navigation Button

This text will appear in this page's navigation button if you include it in a Q-Action mini-web site.

Top

Web Analytics Code

The HTML code snippet for any web analytics service (eg. Google Analytics) that you use to track visitors.

Top

Scan Counter

Leave blank. This will automatically count visits to your action card, excluding ones you make while logged on. Put zero at any time to reset.

Target Website

Put the full URL (including http:// or similar) of the site you want the user to be sent to.

Buy Now Button

Put the full URL of the item on your e-commerce site so that it takes the user directly to the specific product.

Location Coordinates

Put the latitude and longitude or grid ref. For compatibility with future functionality we suggest the following format: 00.00N 000.00E (Note: degrees and decimal degrees, not degrees and minutes). We recommend WGS84 as the datum.

Top

Select Mini Web Page

The drop down boxes list the other pages you have created, using your titles, or the card ID if no title was given. By selecting a page you make it a sub-page of your Mini Web Site. Each action page has a field called 'Label for Navigation Button' so you can specify what text will appear inside its navigation button. If you leave it blank it will use the card title. If that is blank it will use the card's system ID.
