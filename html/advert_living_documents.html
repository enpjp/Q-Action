Living documents that you and others can up-date on the move

Want to be able to access a rota list, an event programme, or a meeting agenda while on location? Want to update it too? Using QR codes from Q-Action you can create this in two steps.

First create the document in a shared workspace such as Google Docs and make it ‘public on the web’ so that anyone with the URL can see it. Editing rights can be shared with others so that specific people can update it.

Then use a Q-Action ‘Goto’ card and put the URL of the document as the re-direct address. As the document is updated your users will instantly see the latest version. If the document changes, for example the next meeting, rota, or event, then simply change the URL in the ‘Goto’ card.

The ‘Goto’ card can point to a photo gallery or a video, so you can produce any kind of on-line content from events and keep it current, even though the QR code is permanently printed on a poster or sign.
